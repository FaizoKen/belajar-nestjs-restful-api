## Create .env file and place your database connection url

```
DATABASE_URL="mysql://root:root@localhost:3306/belajar_nestjs_restful_api_prod"
```

## Migrate

```bash
$ npm run migrate
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
